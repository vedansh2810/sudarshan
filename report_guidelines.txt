ABSTRACT


The project aims to develop a Python script utilizing computer vision techniques to control cursor movement and adjust system volume through hand gesture recognition. Leveraging libraries such as OpenCV, NumPy, Mediapipe, and Autopy, the script provides users with an intuitive and natural interface for interacting with their computers.

The core functionality of the script revolves around detecting and tracking hand gestures captured from the webcam feed. The movement of the index finger is mapped to control the cursor on the screen, allowing users to navigate interfaces effortlessly. Additionally, gestures involving the index and middle fingers simulate left-click actions, enabling users to interact with on-screen elements seamlessly.

To enhance usability, the script incorporates adjustable sensitivity settings for cursor control, ensuring precise and responsive movements tailored to individual preferences. Furthermore, mechanisms are implemented to handle boundary issues, confining cursor movement within the screen boundaries for a consistent user experience.

In addition to cursor control, the script includes a volume controller using virtual mouse movements. By recognizing hand gestures and mapping them to volume adjustments, users can intuitively control system volume without the need for physical input devices.

In some painting software we need to use mouse and keyboard in order to draw or paint on the screen, it has some ups and downs and they also contain limitations. In order to overcome this, we are developing a virtual hand gesture painting system that uses OpenCV and MediaPipe to enable users to draw shapes and write text in a digital environment. The system will use computer vision algorithms to recognize hand gestures and translate them into digital strokes on a virtual canvas.
